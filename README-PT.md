## 🚀 Funcionalidades Free

### ✅ **Recursos Incluídos**
- **Quadro Kanban Interativo**: Organize seus dados em colunas personalizáveis
- **Drag & Drop**: Mova cards entre colunas de forma intuitiva
- **Campos Suportados**:
  - 📊 **Colunas do Kanban** (obrigatório)
  - 📝 **Títulos dos Cards** (obrigatório)
  - 👤 **Responsáveis** (opcional)
  - 🆔 **ID Único** (opcional)
  - 📅 **Última Atualização** (opcional)
  - ⭐ **Prioridade** (opcional com badges coloridos)
  - 🎨 **Ícone (URL)** (opcional - suporte a emojis e imagens)

### 🎨 **Personalização de Layout**
- **Largura das Colunas**: Ajuste em pixels
- **Espaçamento dos Cards**: Controle o espaçamento vertical
- **Espaçamento das Colunas**: Controle o espaçamento horizontal
- **Banner Informativo**: Pode ser ocultado nas configurações

### 📊 **Badges de Prioridade**
- 🔴 **Critical** - Vermelho
- 🟡 **High** - Amarelo  
- 🟢 **Medium** - Verde
- ⚪ **Low** - Branco

### 📈 **Limites da Versão Free**
- **Máximo de 80 cards** por visualização
- Aviso automático quando o limite é atingido
- Sugestão para filtrar dados para melhor performance

## 📦 Instalação

1. **Baixe o arquivo**: `kanbanReact3783AB73E36E439FA1F672E13E7CFE7E.1.0.0.0.pbiviz`
2. **No Power BI Desktop**:
   - Vá em `Arquivo` → `Importar` → `Visual do Power BI`
   - Selecione o arquivo `.pbiviz` baixado
   - O visual aparecerá no painel de visualizações

3. **Configure os campos**:
   - Arraste suas colunas para os campos correspondentes
   - Configure o layout através do painel de formatação (🎨)

## 🔧 Como Usar

### **1. Preparação dos Dados**
Certifique-se de que seus dados tenham pelo menos:
- Uma coluna para **status/etapa** (ex: "A Fazer", "Em Progresso", "Concluído")
- Uma coluna para **título/descrição** da tarefa

### **2. Configuração no Power BI**
1. Adicione o visual à sua página
2. Configure os campos obrigatórios:
   - **Colunas do Kanban**: Campo que define as etapas
   - **Títulos dos Cards**: Campo com o nome/descrição das tarefas
3. Configure campos opcionais conforme necessário

### **3. Personalização**
- Use o painel de formatação (🎨) para ajustar layout
- Configure cores e espaçamentos conforme sua marca
- Ative/desative o banner informativo

## 💡 Dicas de Uso

- **Performance**: Para melhor performance, filtre seus dados para mostrar apenas as tarefas relevantes
- **Prioridades**: Use valores como "Critical", "High", "Medium", "Low" para badges automáticos
- **Ícones**: Use emojis (🔧, 📊, 🎯) ou URLs de imagens para ícones personalizados
- **Responsáveis**: Campo de texto livre para nomes ou iniciais
