# 📋 Notas de Versão - Kanban Visual Free

## 🎉 v1.0.0 - Lançamento Inicial (30/06/2025)

### ✨ Novidades
Primeira versão pública do Kanban Visual Free para Power BI!

#### 🚀 Funcionalidades Principais
- **📋 Quadro Kanban Interativo**: Visualização de cards organizados por colunas
- **👥 Campo Responsáveis**: Exibição do responsável por cada tarefa
- **🏷️ Sistema de Prioridades**: Badges coloridos automáticos (Alta=Vermelho, Média=Amarelo, Baixa=Verde)
- **🎨 Suporte a Ícones**: Compatibilidade com emojis e URLs de imagem
- **📐 Layouts Flexíveis**: Suporte para 3, 4 e 5 colunas
- **⚙️ Personalização**: Controle de largura das colunas e espaçamentos

#### 🔧 Campos Suportados
- **Colunas do Kanban** (Obrigatório): Define as colunas do quadro
- **Títulos dos Cards** (Obrigatório): Texto principal de cada card
- **Responsáveis** (Opcional): Nome da pessoa responsável
- **ID Único** (Opcional): Identificador único para rastreamento
- **Última Atualização** (Opcional): Data da última modificação
- **Prioridade** (Opcional): Nível de prioridade com cores automáticas
- **Ícone (URL)** (Opcional): Emoji ou link para imagem

#### 🎯 Versão Gratuita
- **Limite de 80 Cards**: Versão free suporta até 80 registros
- **Banner Informativo**: Informações sobre doações e contato (pode ser ocultado)
- **Funcionalidades Completas**: Todos os recursos visuais disponíveis

### 🔧 Configurações Técnicas
- **Compatibilidade**: Power BI Desktop e Power BI Service
- **Tecnologias**: React + TypeScript
- **Tamanho do Arquivo**: ~1.2MB
- **Performance**: Otimizado para até 80 cards

### 📊 Dados de Exemplo
- Incluído arquivo `Testes.pbix` com dados de demonstração
- Exemplos de diferentes cenários de uso
- Configurações pré-definidas para teste rápido
