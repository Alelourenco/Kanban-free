# 📋 Free Kanban Visual for Power BI

An interactive Kanban board visual for Power BI that allows you to organize your data with drag-and-drop functionality.

## ✨ Free Features

### 🎯 Main Features
- **📋 Interactive Kanban Board**: Organize cards by customizable columns
- **👥 Responsible Field**: Display who is responsible for each task
- **🏷️ Priority Badges**: Colored badges for different priorities
- **🎨 Icon Support**: Use emojis or image URLs
- **📐 Customizable Layouts**: Choose between 3, 4, or 5 columns
- **🔢 80 Cards Limit**: Free version supports up to 80 cards

### 🎨 Supported Fields
- **Kanban Columns** (Required): Defines the board columns
- **Card Titles** (Required): Main title of each card
- **Responsible** (Optional): Person responsible for the task
- **Unique ID** (Optional): Unique identifier for each card
- **Last Updated** (Optional): Date of last modification
- **Priority** (Optional): High, Medium, Low (with automatic colors)
- **Icon (URL)** (Optional): Emoji or image URL

## 🚀 Quick Installation

### 1. Download the Visual
1. **Download the file**: `kanbanReact3783AB73E36E439FA1F672E13E7CFE7E.1.0.0.0.pbiviz`
2. **In Power BI Desktop**:
   - Go to "Insert" → "More visuals" → "Import visual from file"
   - Select the downloaded `.pbiviz` file
   - The visual will appear in the visualizations panel

### 2. Basic Setup
1. **Drag fields** to the corresponding areas in the visual
2. **Configure layout** in the formatting panel
3. **Adjust banner** if needed (can be hidden)

## 🎯 How to Use

### Minimum Configuration
```
Kanban Columns: Status (e.g., "To Do", "In Progress", "Done")
Card Titles: Task Name
```

### Complete Configuration
```
Kanban Columns: Status
Card Titles: Task Name
Responsible: Person Name
Priority: High/Medium/Low
Icon (URL): 🔥 or https://example.com/icon.png
Unique ID: Task ID
Last Updated: Modification Date
```

## ⚙️ Customization Options

### Kanban Layout
- **Column Width**: Adjust in pixels (300-600px)
- **Card Spacing**: Space between cards (8-20px)
- **Column Spacing**: Space between columns (16-40px)
- **Show Free Banner**: Display/hide informative banner

### Priority Colors (Automatic)
- 🔴 **High**: Red
- 🟡 **Medium**: Yellow
- 🟢 **Low**: Green

## 🔧 Support and FAQ

### 🆘 Common Issues
- **Visual doesn't appear**: Check if the `.pbiviz` file was imported correctly
- **Cards don't show**: Make sure you have required fields (Columns and Titles)
- **80 cards limit**: Free version is limited to 80 records
- **Banner always visible**: Disable in "Show Free Banner"

### 📝 Report Issues
1. [Create an issue](https://github.com/Alelourenco/Kanban-free/issues/new) on GitHub
2. Describe the problem in detail
3. Include screenshots if possible
4. Mention your Power BI version
